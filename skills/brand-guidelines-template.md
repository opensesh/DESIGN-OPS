# Brand Guidelines Template

A customizable brand identity skill that users configure with their own brand standards. This skill serves as a template for creating organization-specific brand guidance.

## Purpose

Enable Claude to apply consistent brand standards across all creative work. Users should customize this template with their own brand values, colors, typography, and voice guidelines.

## When to Activate

Use this skill when:
- Creating any branded content or communications
- Reviewing work for brand compliance
- Making design decisions that affect brand consistency
- Generating copy that needs to match brand voice

## Questions to Ask

When unsure about a brand decision:
1. Does this feel like [Brand Name]?
2. Would this be recognizable as us without the logo?
3. Does this align with our core values?
4. Is this how we want to be perceived?
